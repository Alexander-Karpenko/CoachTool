# Training System

Веб-приложение для автоматизации формирования и анализа спортивных тренировочных программ.

---

# Описание проекта

Система предназначена для тренеров и позволяет:

- формировать тренировочные программы;
- управлять подопечными;
- отслеживать тренировочные показатели;
- анализировать прогресс спортсменов;
- строить графики и отчеты;
- рассчитывать тренировочные объемы и интенсивность.

Функциональные требования
1. Управление пользователями

Система должна обеспечивать:

регистрацию тренера;
авторизацию тренера;
восстановление пароля;
редактирование профиля тренера.
2. Управление подопечными

Тренер должен иметь возможность:

создавать карточку подопечного;
редактировать данные подопечного;
удалять подопечного;
просматривать список подопечных.
Карточка подопечного должна содержать:
имя;
возраст;
рост;
вес;
спортивную квалификацию;
дату начала тренировок;
контактную информацию.
3. Управление упражнениями

Система должна:

предоставлять список упражнений;
позволять искать упражнения;
позволять добавлять новые упражнения;
хранить категории упражнений.
Для упражнения хранится:
название;
группа мышц;
тип упражнения;
единица измерения нагрузки.
4. Формирование тренировочной программы

Тренер должен иметь возможность:

создавать программу на неделю;
копировать программу предыдущей недели;
редактировать программу;
удалять программу.
Для каждого упражнения указываются:
количество подходов;
количество повторений;
рабочий вес;
процент от максимума;
комментарии тренера.
5. Работа с максимумами

Система должна:

хранить максимальные показатели (1RM);
автоматически рассчитывать процент нагрузки;
обновлять максимумы вручную или автоматически.
6. Аналитика и отчеты

Система должна формировать:

отчет по выбранному периоду;
отчет по упражнениям;
отчет по подопечному;
график изменения нагрузок;
график изменения максимумов;
статистику тренировочного объема.
Аналитика должна включать:
пиковую нагрузку;
среднюю нагрузку;
суммарный объем;
изменение нагрузки в процентах;
динамику прогресса.
7. Отправка тренировочной программы

Система должна:

экспортировать программу в изображение или PDF;
отправлять программу:
по электронной почте;
через мессенджер.
Дополнительные функции
1. Автоматический расчет тренировочного объема

Формула:

V=Σ(weight×reps×sets)

Используется для анализа общей тренировочной нагрузки.

2. Расчет процента от максимума

Формула:

P=(workingweight/1RM)×100

Используется для автоматического определения интенсивности нагрузки.

3. Автоматическое определение прогресса

Система должна анализировать:

рост максимума;
увеличение тренировочного объема;
изменение интенсивности.
4. Графики и визуализация

Система должна поддерживать:

графики нагрузки;
графики прогресса;
диаграммы объемов;
визуализацию статистики.
5. История изменений программы

Позволяет:

отслеживать изменения недельных программ;
откатывать изменения;
анализировать тренировочные циклы.
